Prueba tecnica de UDNET

programa hecho con react y boostrap para realizar las decoraciones finales
Arquitectura basada en componentes

Para iniciar descargaremos node.js y despues de la instalacion seguiremos los pasos a continuacion dependiendo de si tienes o no visual 
studio code.

iniciando el programa descargue el repositorio y una vez descargado, descomprima el archivoy 
abra la consola de windows oprimiendo las teclas:
Windows + R y poniendo cmd en el buscador ahi ingrese los siguientes comandos
1. cd ..
2. cd Usuario
3. cd Downloads
4. cd Prueba-Tecnica-UDNET-main
5. cd conversor-divisas
6. npm run dev
7. ctrl + click en el link que aparece
8. ya etsas dentro :D
